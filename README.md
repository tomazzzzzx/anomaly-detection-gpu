# Anomaly Detection Gpu

Real-time anomaly detection on GPU for streaming time-series data.

## Algorithms
- GPU-accelerated Isolation Forest
- Streaming PCA for high-dimensional data
- LSTM-based autoencoder anomaly scorer
- Sliding window statistics

## Throughput
| Method | Records/sec | Latency |
|--------|------------|---------|
| Isolation Forest | 2.1M | 0.5ms |
| LSTM Autoencoder | 890K | 1.2ms |
